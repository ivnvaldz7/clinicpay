# ClinicPay Frontend

This package contains the React frontend for ClinicPay.

## Stack

- React
- Vite
- React Router
- Zustand
- Tailwind CSS
- Radix UI
- Recharts

## Run Locally

```bash
npm install
npm run dev
```

## Quality

```bash
npm run lint
```

For the full project overview, architecture, and setup instructions, see the root [README](../README.md).
